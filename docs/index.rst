NLTools
=======

`NLTools <https://github.com/ljchang/neurolearn>`_ is a Python package to analyze imaging data.  It is the analysis engine powering `neuro-learn <http://neuro-learn.org>`_ There are tools to perform preprocessing, univariate GLMs, an predictive multivariate modeling.  It is based off of Tor Wager's `object-oriented Matlab toolbox <https://github.com/canlab/CanlabCore>`_ and leverages much code from `nilearn <http://nilearn.github.io/>`_ and  `scikit-learn <http://scikit-learn.org>`_
